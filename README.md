# minesweeper

Консольный сапёр на Python.

## Запуск

Скачайте репозиторий и перейдите в папку проекта:

~~~shell
git clone https://github.com/Masynchin/minesweeper.git
cd minesweeper
~~~

Чтобы запустить игру, выполните:

~~~shell
python main.py
~~~

Чтобы узнать правила игры, выполните:

~~~shell
python main.py --help
~~~
